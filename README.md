# Userscripts

A collection of browser userscripts.

## Scripts

### Threads Hide Login Overlay

Remove the login wall, header bar, bottom navigation, and "Get app" CTAs from [Threads](https://www.threads.com) so you can read posts without signing in.

**Install:** [threads-hide-login.user.js](threads-hide-login.user.js)

**Features:**

- Hides the login/signup overlay dialog (hero text + backdrop)
- Hides the top header bar (`#barcelona-header`)
- Hides the bottom navigation bar (`nav`)
- Hides standalone CTA buttons (Log in, Get app, Open app, etc.)
- Removes scroll lock imposed by the overlay
- Resets `--header-height` so content moves to the top after header removal

**Compatibility:** Tampermonkey, Greasemonkey, Violentmonkey
